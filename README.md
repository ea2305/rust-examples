# Rust Lang : Basic Examples

This repo provides simple examples described in the rust lang book. Rigth now I have implement the following examples:

* Hello world
* Types
* Arrays
* Strings
* Simple Game example
* Loops (loop, for)
* Branches (If, match)


## Requirements

* Rust lang compile (1.*)
* Rust Cargo (1.37.0)
